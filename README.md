# Auto-Saving-Piggy-Bank-Self-Growing-Wallet-
A decentralized piggy bank smart contract that lets the owner deposit ETH, which stays locked for 30 days. After the lock period, the owner can withdraw the amount with simple interest. It also includes an emergency withdrawal option that allows the owner to take out funds anytime without earning interest.
Project Vision

To create a secure, automated on-chain savings system that helps users develop consistent saving habits while rewarding them with interestâ€”without relying on intermediaries.

# Key Features

Accepts ETH deposits directly into the contract

Automatically locks funds for 30 days

Rewards users with 5% simple interest on withdrawal

Emergency withdrawal without interest

Transparent, immutable savings logic on the blockchain

# Future Scope

Tier-based or dynamic interest rates

Deposit history tracking
